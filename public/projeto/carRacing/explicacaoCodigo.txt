Main.js:

 - O Phaser é uma ferramenta para criar jogos com JavaScript.

 - Na POO, a gente cria classes, que são como modelos para fazer objetos.
   Exemplo: class Cachorro é um modelo de cachorro. Depois você pode criar vários cachorros com esse modelo. 


Classes e cnonstrutores do POO:
Você criou uma classe chamada CenaPrincipal, que é uma cena do jogo.

- No Phaser, uma "Cena" é como uma tela. Pode ser a tela inicial, tela do jogo, tela de pausa, etc.

- extends Phaser.Scene: significa que essa cena usa tudo que o Phaser já tem pronto.

- super({key: 'PrincipalCena'}): você dá um nome para a cena, nesse caso "PrincipalCena", para poder chamar ela depois.


Configuração do Jogo

- type: Phaser.AUTO: o Phaser escolhe automaticamente se vai usar WebGL ou Canvas (formas de mostrar o jogo).

- width e height: largura e altura da tela (usando as constantes de antes).

scale:

- mode: Phaser.Scale.FIT: o jogo se ajusta automaticamente ao tamanho da tela do jogador.

- autoCenter: Phaser.Scale.CENTER_BOTH: o jogo fica centralizado no meio da tela.

- scene: são as cenas que seu jogo terá. Aqui você colocou duas: CenaPrincipal e CenaPausa.
